[] // 匹配中括号中的任意一个字符
[a-z0-9A-Z] // 匹配大小写字母 0-9数字 中任意一个
[^aaa]  // 查找不再方括号内的字符
[a-zA-Z] == [A-z]
/a|b/ // 匹配a 或者 b

. 查找单个字符 匹配所有字符 除了换行和行结束符
\w 查找单词字符 包括字母 数字 下划线
\W 查找非单词字符 包括字母 数字 下划线

\d 查找数字字符
\D 查找非数字字符

\s 查找空白字符（空格 Tab 换行 回车）只是匹配一个空白字符
\S 查找非空白字符


\b 匹配单词边界  var str= "Hello Hunger" str.matach(/\bHung/) Hung 前面有空白或者字符串
第一个单词
\B 匹配非单词边界

n+ 匹配n前面的字符 至少一次 或多次
n* 匹配0个或者多个
n? 0个或者1个

n{x} 匹配包含X个N 的序列字符串
n{x,y} 匹配X或Y个N 的序列字符串 默认是贪婪模式
n{x,} 匹配至少包含X个N 的序列字符串

n$ 必须以n作为结尾
^n 匹配以n开头的字符串
